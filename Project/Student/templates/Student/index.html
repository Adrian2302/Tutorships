Kaizen es un sistema que une a los estudiantes y los profesores.

¿Pero cómo se usa?

Antes de usar nuestro sistema debe de estar registrado, si no lo esta procesada a registrarse en el botón de acceder.
Una vez registrado podrá buscar y agendar una tutoría, para esto puede buscar el curso que desea.
Una vez encontrado el curso, podrá agendar la tutoría con un profesor y con el horario que más le convenga. No obstante, cada profesor da las tutorías de una manera diferente, por lo que se recomienda que lea bien las diferentes caracteristicas del profesor.
Luego de esto podrá ver el estado de la tutoría, si está pendiente, aceptada o rechazada. Esto lo puede hacer en el apartado de mis tutorías.
Cuando una tutoría se realiza, se le pide una retroalimentación para el profesor y para los próximos estudiantes.
